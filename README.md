# Todo Tracker REST API

A robust RESTful API built to manage personal task lists. This project was developed to gain hands-on experience and a deeper understanding of backend development using Python, Django, and the Django REST Framework (DRF).

## 🚀 Project Overview

The Todo Tracker API allows users to create accounts, log in securely, and manage their own daily tasks. It is designed with data privacy in mind—users can only perform CRUD (Create, Read, Update, Delete) operations on their own personalized to-do lists. 

Additionally, the API is fully documented using Swagger, providing an interactive interface to test and explore the endpoints.

## ✨ Features

* **User Authentication:** Secure registration and login functionality.
* **Personalized Data:** Users have exclusive access to their own to-do items. 
* **Full CRUD Operations:** * Create new to-dos.
  * Read/view specific or all personal to-dos.
  * Update existing to-dos (mark as complete, edit text, etc.).
  * Delete to-dos.
* **Interactive API Documentation:** Integrated Swagger UI for easy endpoint testing and exploration.

## 🛠️ Tech Stack

* **Language:** Python
* **Framework:** Django
* **API Toolkit:** Django REST Framework (DRF)
* **Documentation:** Swagger (OpenAPI)
